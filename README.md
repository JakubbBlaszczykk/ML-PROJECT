## Machine Learning Team Project

In this repository we are working on the group project for the course Machine Learning.

## Project Description:
The goal of the project is to create the search engine base on the IMDb Dataset avable on https://developer.imdb.com/non-commercial-datasets. The repository contains the Data Cleaning and Preprocessing, Exploratory Data Analysis, prepared Custom Transformers and Machine Learning Models. The project is implemented in Python with usage of several libraries.
